# 👟 Trending Shoes

A simple E-Commerce website built using PHP, MySQL, HTML, CSS and Bootstrap.

## ✨ Features

- User Registration
- User Login & Logout
- Session Authentication
- Product Listing
- Add to Cart
- Increase / Decrease Quantity
- Remove from Cart
- Buy Product
- Purchase Success Alert
- Grand Total Calculation
- Continue Shopping

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- XAMPP

## 🗄️ Database

Import the `e-com.sql` file into MySQL before running the project.

## 🚀 How to Run

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project to `htdocs`.
4. Import `e-com.sql` into phpMyAdmin.
5. Open:
   ```
   http://localhost/e-commerce
   ```
